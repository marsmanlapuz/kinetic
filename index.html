Problem 2 Distance Between Two Points Calculator









Example to get answer.
Sample Distance between two trees there are two tree in a park Point A. Point B. The coordinates of the tree is (x¹,y¹) = (2,3) Point A. and the coordinates of the second tree is (x2,y2) = (7,8) Point B.to compute the distance Point A. Point B. solution: D = (X²-x¹)² + (y²-y¹)² Subtitute the given values into the formula: D = (7-2)² + (8-3)² Simplify each part: 7-2 = 5, so (7-2)² = 5² = 25 8-3 = 5, so (8-3)² = 5² = 25 no we have: D = 25 + 25 = /50 Take the square root of 50: D = 7.07 Units.
